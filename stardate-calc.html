STARDATE CALCULATOR

The code for the following tools was created by Andreas Schmidt. The internal math provided by Andreas Schmidt does account for leap years.
The version below has been modified for use by the 38th Fleet 'Argo' to reflect the in-game time for Star Trek Online with a +399 year boost to reflect the 2010 release with in-game year of 2409, and for clarity and formatting.

Stardate to Calendar Date

Use this tool to converts stardates to in-character calendar dates. Type in the stardate you want to convert and press the "Convert" button. The calendar date will be displayed in the box below.

Note: Stardates prior to January 1, 2333 (Stardate 10000) will not convert properly.

IRL Calendar Date to Stardate

Use this tool to create a stardate based on the date in real life (IRL). Select the IRL date that you want to convert and press the "Convert" button. The stardate will be displayed in the box below. The default values will give you the current stardate (based on your computer's system clock). Note: IRL dates prior to January 1, 1934 (Stardate 10000) will not convert properly.

This converter is NOT for in-character calendar dates; if you are entering in-character years (ie. 2415), use the tool below.
